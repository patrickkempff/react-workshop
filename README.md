# react-workshop

Oefeningen van mijn React Workshop

## Getting started

```bash
$ git clone https://github.com/patrickkempff/react-workshop.git
$ cd react-workshop
$ npm install
$ npm start
$ surf to http://127.0.0.1:8080
```
